> **Требования** 🛠️
> - Python version: `3.10`
> - AIOgram version: `3.x`
>
> 📝 Создайте файл `.env` и добавьте туда токен вашего бота.

---

### 📦 Установка модулей
```bash
pip install aiogram motor pymongo dnspython -U
```

### 🚀 Запуск проекта
```bash
git clone https://github.com/Fsoky/anonimchatbot-aiogram3.git
```
```bash
cd anonimchatbot-aiogram3/src
```
```bash
python __main__.py
```
